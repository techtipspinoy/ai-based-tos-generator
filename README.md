# A free tool for Philippine teachers to create standards-based assessments.

## Generate DepEd-compliant Table of Specifications (TOS) and quizzes/exams aligned with the Most Essential Learning Competencies (MELCs) for Grades 7–10.

- Fully offline-friendly
- One-click Word (.docx) export
- Optional AI-powered question generation (Ollama or OpenAI)
- Preloaded MELC database (Math, Science, English, AP, ESP, TLE, MAPEH)

## Compliance & Licensing

### Aligned with:
- DepEd Order No. 012, s. 2023 (MELCs)
- DepEd DM-CI-2023-00671 (Assessment Guidelines),
- RA 8293 (Intellectual Property Code) — No verbatim MELC reproduction beyond short competency statements.

## Quick Start
- git clone https://github.com/techtipspinoy/ai-based-tos-generator.git
- cd ai-based-tos-generator
- pip install -r requirements.txt
- streamlit run app.py
